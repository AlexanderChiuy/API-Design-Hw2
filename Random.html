selectRandom(choice)
A function that will return a random item from the inputted "choices" list.
 
Args:
 - choices: a list of possible items to choose from. This list can compose of 
            different typed items or all the same type. 
            Example: 
                choices = ["a", 1, "b", dict()] or ["hello", "hi", "howdy"]
            Exception:
                ValueError for None or empty list 
 
Returns: an item randomly from the "choices" list. Return type can be any type in the "choices" list
 
Example Use Case: 
   randomColor = selectRandom(["red", "yellow", "blue"])
